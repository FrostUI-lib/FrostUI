## TiltCard Component

The `TiltCard` component provides an engaging way to display information with a subtle 3D tilt effect that responds to mouse movement. It's useful for highlighting content and adding a touch of interactivity to your UI.

```bash showLineNumbers: true
npx shadcn-ui@latest add https://Frostui.in/r/TiltCard.json
```
