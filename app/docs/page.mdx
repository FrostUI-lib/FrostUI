## Accordion Component

The `Accordion` component provides a way to toggle the visibility of content within a container. It's useful for organizing information and presenting it in a user-friendly way.

### Basic Usage

<div
  style={{
    border: '1px solid #e5e7eb',
    borderRadius: '0.75rem',
    padding: '1.5rem',
    marginTop: '2rem',
    marginBottom: '2rem',
  }}
>
  <Tabs items={['Preview', 'Code']}>
    <Tabs.Tab>
      <div style={{ display: 'flex', justifyContent: 'center' }}>
        <BasicAccordion />
      </div>
    </Tabs.Tab>
    <Tabs.Tab>
      ```jsx showLineNumbers: true,
      import { Accordion } from '@/components/Accordion/accordion';

      export function BasicAccordion() {
        return (
          <Accordion label="Basic Accordion" variant="basic">
            This is a basic accordion. Itâ€™s useful for displaying concise information that can be expanded on demand.
          </Accordion>
        );
      }
      ```
    </Tabs.Tab>
  </Tabs>
</div>
