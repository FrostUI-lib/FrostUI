## FancyFloatingInfo Component

The `FancyFloatingInfo` component provides a way to display contextual information, warnings, or errors in a floating panel that appears on hover or focus of a trigger element.


## Installation via CLI

```bash showLineNumbers: true,
npx shadcn-ui@latest add https://frostui.in/r/Button.json
```

### Variant Types

The `FancyFloatingInfo` component supports different visual variants to convey the nature of the information.

#### Info Tooltip

Used to display general information or helpful tips.

<div
  style={{
    border: '1px solid #e5e7eb',
    borderRadius: '0.75rem',
    padding: '1.5rem',
    marginTop: '2rem',
    marginBottom: '2rem',
  }}
>
  <Tabs items={['Preview', 'Code']}>
    <Tabs.Tab>
      <div style={{ display: 'flex', justifyContent: 'center', gap: '1rem' }}>
        <InfoTooltip />
      </div>
    </Tabs.Tab>
    <Tabs.Tab>
        ``` jsx showLineNumbers: true
        <FancyFloatingInfo
            label="Info Tooltip"
            content={{
            title: "Info Tooltip",
            description: "This is an info tooltip that provides additional information about a specific element or action.",
            }}
            variant="info"
        />
        ```
    </Tabs.Tab>
  </Tabs>
</div>
