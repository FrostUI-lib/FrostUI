## Loader Component

The `Loader` component provides a visual cue to users that an operation is in progress. It comes in various sizes, types, and speeds to suit different contexts within your application.

### Basic Usage

<div
  style={{
    border: '1px solid #e5e7eb',
    borderRadius: '0.75rem',
    padding: '1.5rem',
    marginTop: '2rem',
    marginBottom: '2rem',
  }}
>
  <Tabs items={['Preview', 'Code']}>
    <Tabs.Tab>
      <div style={{ display: 'flex', justifyContent: 'center' }}>
        <BasicLoader />
      </div>
    </Tabs.Tab>
    <Tabs.Tab>
      ```jsx showLineNumbers: true,
      <Loader />;
      ```
    </Tabs.Tab>
  </Tabs>
</div>

## Installation via CLI

```bash showLineNumbers: true,
npx shadcn-ui@latest add https://Frostui.in/r/Loader.json
```
